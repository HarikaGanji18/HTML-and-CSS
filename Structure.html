Basic structure of HTML

An html document starts and ends with html tags. These tags tells the browser that the entire document is composed in HTML.
Inside these two html tags, the document is split into two sections.

  1. Header/Head Section (Optional)
  2. Body Section (Mandatory)

Syntax

//consider '/' as less than symbol and '\' this as greater than symbol//

/!Doctype html\ ----Present Version of the HTML
/html\

//HEAD SECTION

/head\
/title\ Structure of an HTML Document //title\
//head\


//BODY SECTION

/body\

/p\content...//p\
/b\content...//b\
/h3\content...//h3\
.............
...........
//body\
//html\
